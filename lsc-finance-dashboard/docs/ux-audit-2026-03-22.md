# UX Audit — March 22, 2026

## Critical Issues (P0-P1)

### 1. Race/Season Selection — No Smart Defaults
Every race-dependent page requires manual selection. Should auto-select the next upcoming race.
**Pages:** Races, Costs, Payments, Expense Management

### 2. Information Overload — Endless Scrolling
Every page has too much explanatory text, too many process-step cards, and AI comments taking 20-30% of page height.
**Fix:** Tooltips instead of paragraphs. Collapse AI insights. One-sentence directives only.

### 3. Table Column Overflow
Tables with 7-9 columns overflow on laptops.
**Pages:** Expense management (8 cols), Documents (7 cols), Payments (6 cols)
**Fix:** Show critical columns only, use expandable rows for detail.

### 4. No Progress Indicators on Multi-Step Pages
Costs, Payments, Commercial Goals use tabs but no visual progress.
**Fix:** Numbered step indicators with current step highlighted.

### 5. Expense Review Page — Clunky
- Filter form (4 dropdowns) always expanded
- Budget rules table too wide
- Duplicate process-step info blocks
- No default race selection
**Fix:** Collapse filters after applied, reduce table columns, show one info block.

### 6. Invoice → Expense → Reimbursement Flow Not Visible
Data flows across 4 pages with no breadcrumb showing lifecycle.
**Fix:** Add flow indicator: "Invoice uploaded → Analyzed → Expense report → Payment queued"

### 7. Metric Card Subtitles Too Verbose
Every metric has 1-2 sentence explanations that clutter the view.
**Fix:** Show label + value only. Tooltip for context.

### 8. Status Naming Inconsistent
Different status labels across pages (submitted, pending_review, analyzed, issued, etc.)
**Fix:** Unified status system with consistent colors.

### 9. AI Comments Take Too Much Space
3-4 process-step blocks per page explaining what data means.
**Fix:** Show 1 top insight inline, rest in collapsible panel or /ai-analysis page.

### 10. Empty States Not Actionable
Tables show "No X yet" without guiding users to create first record.
**Fix:** Add CTA links in empty states.
